class crpropa::HDF5Output

Output to HDF5 Format.

HDF5 structure: ``` HDF5 "FILENAME.h5" { GROUP "/" { DATASET "OUTPUTTYPE" { DATATYPE H5T_COMPOUND { ... } DATASPACE SIMPLE { ( 1 ) / ( H5S_UNLIMITED ) } DATA { ... } ATTRIBUTE "Version" { DATATYPE H5T_STRING { STRSIZE 100; STRPAD H5T_STR_NULLTERM; CSET H5T_CSET_ASCII; CTYPE H5T_C_S1; } DATASPACE SCALAR DATA { (0): "VERSION" } } } } } ```
